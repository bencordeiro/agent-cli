AI-Agent CLI (WSL Runner) Instructions
======================================

Overview
--------
This CLI runs commands inside WSL (your default Linux distro) and can also
use your vLLM server to decide which tools to call.

Files
-----
- agent_cli.py : main CLI
- config.toml  : defaults + approval mode + LLM config

Prereqs
-------
- Windows with WSL installed
- WSL Linux tools: curl, ping (iputils), nslookup (dnsutils), nmap (optional)
- Python 3.11+ on Windows
- vLLM running at your configured base URL with tool-calling enabled

Approval Mode
-------------
Default is prompt for each command. To auto-approve:

  config.toml:
    [approval]
    mode = "auto"

Or per run:
  python agent_cli.py --auto-approve <command>

Basic Commands
--------------
1) Weather (reads your configured weather URL)
  python agent_cli.py --auto-approve weather

Example output:
  Weather:
  - Temperature: 23.9 C
  - Humidity: 74.0%
  - Light: 12
  - Wi-Fi RSSI: -53 dBm

2) Site check (reads your configured site URL)
  python agent_cli.py --auto-approve site-check

Example output:
  OK: <site URL> returned 200

3) Ping
  python agent_cli.py --auto-approve ping YOUR_HOST

4) Nslookup
  python agent_cli.py --auto-approve nslookup example.com

5) Nmap
  python agent_cli.py --auto-approve nmap YOUR_HOST -- -sV -p PORTS

6) Run arbitrary command in WSL
  python agent_cli.py --auto-approve run -- df -h

7) Check servers (site check + weather site status + ping)
  python agent_cli.py --auto-approve check-servers

Example output:
  Results:
  - Site OK (HTTP 200): <site URL>
  - Weather site OK (HTTP 200): <weather URL>
  - Ping OK: YOUR_HOST

8) Chat mode (multi-turn, tool access each turn)
  python agent_cli.py --auto-approve chat

Chat commands:
  /tools  (list available tools)
  /exit   (quit)

Save conversation history (JSON):
  python agent_cli.py --auto-approve chat --save-history history.json
  python agent_cli.py --auto-approve ask "..." --save-history history.json

LLM Agent Mode
--------------
Ask the LLM to use tools and summarize results:

  python agent_cli.py --auto-approve ask "Check my servers and what is the temperature right now"

Example output:
  Based on the output, here are the results:
  Server Status: OK (HTTP 200)
  Ping: OK (4/4)
  Temperature: 23.9 C

Notes
-----
- All execution happens in WSL via wsl.exe.
- Weather parsing looks for .box elements containing .label and .value.
- File tools operate only inside the repo workspace.
- Deleting or overwriting files always requires confirmation.
